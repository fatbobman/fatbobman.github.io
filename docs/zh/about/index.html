Hi, 我是徐杨

在互联网上我用东坡肘子（ Fatbobman ）的笔名来发表文章。

一个偶然的机会让我迈入了苹果生态开发者的行列。最初本打算作为一种恢复身体的手段（ 改善因药物创伤导致迟钝的大脑 ）以及开发一个自己用的 App，没想到投入的时间越来越多，如今俨然已成为了生活中的重要组成部分。

创建这个博客的初衷是为了记录我在学习过程中碰到的问题，随着读者的增多，现在也会有意识地写一些适合更多人阅读的文章。

目前我主要专注于 Swift、SwiftUI、Core Data 和 SwiftData 等技术领域，随着我的知识不断拓展，我期待未来能涉猎更多的技术范畴。
